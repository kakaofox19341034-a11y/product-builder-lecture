# Animal Face Test (Dog vs Cat)

## Overview

This is a web application that uses a Teachable Machine AI model to classify whether a user looks more like a Dog or a Cat based on an uploaded photo.

## Features

*   **AI Classification:** Uses a pre-trained Teachable Machine model.
*   **Image Upload:** Supports file selection and drag-and-drop.
*   **Real-time Analysis:** Displays confidence levels for each animal type.
*   **Responsive Design:** Works on both mobile and desktop.
*   **Dark Mode:** Supports a togglable dark theme.
*   **Community Features:** Includes a partnership inquiry form and a Disqus comment section.

## Model Details

*   **Model URL:** `https://teachablemachine.withgoogle.com/models/ay6JupUUH/`
*   **Classes:** Dog, Cat (and potentially others depending on the model).

## Technical Stack

*   **Frontend:** HTML5, CSS3, JavaScript.
*   **AI/ML:** TensorFlow.js, Teachable Machine Image Library.
